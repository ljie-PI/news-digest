## Talkie: a 13B vintage language model from 1930

### 事件背景

talkie-lm.com 发布 Talkie——一个仅用 1930 年前文本语料训练的 13B 参数语言模型。它没有 ChatGPT 时代的现代偏见，也不知道二战后的世界，专门用来"和过去的人对话"。模型权重开放下载，HN 评分 763。提交标语写道："Have you ever daydreamed about talking to someone from the past?"——评论区 teleforce 立刻翻出 1985 年乔布斯的访谈视频，乔布斯当年描述未来的电脑会让人们与亚里士多德对话，"现代 LLM 就是当年的愿景"。

### 核心机制

Talkie 走的是"语料切片 + 从零训练"路线，与系统 prompt 装老相不同。HN 用户 walrus01 评论："你也可以拿 35B 或 122B 模型加一个详细 system prompt 让它说维多利亚-爱德华时期英语风格"——这正是 Talkie 想避开的方案：**词汇量、世界观、隐喻系统**都来自当年文本，模型不知道飞机意味着什么，也没有现代政治观念。Talkie 提到的相关项目 TimeCapsuleLLM（github.com/haykgrigo3）走完全相同思路，专门"减少现代偏见、模拟特定时期的声音、词汇与世界观"。

### 社区热议与争议点

讨论分四个流派。**实用派（aftbit）**抱怨："只有 20GB VRAM，得换更强的机器了"——13B 模型对家用硬件门槛偏高。**质疑派（yesitcan）**调侃："Vintage 是个有趣的称呼，它跑在真空管硬件上吗？"——指出这种"vintage"只是语料的，不是计算的。**学术派（sega_sai）**提出最有深度的问题："研究这种模型能否独立推导出广义相对论之类的结果会很有意思"——这本质是反事实科学史实验。**反鹦鹉派（twoodfin）**认为 Talkie 给出的 Python 示例（一个 1930 年模型怎么处理"Python"这个词）是"对那些还在嘲笑 LLM 是随机鹦鹉的人的有力回击"。

### 行业影响与未来展望

Talkie 代表"语料考古学 LLM"细分赛道：同类项目还有专门的维多利亚伦敦 LLM、古希腊文献 LLM 等。**应用场景**包括历史教学、文学研究、历史小说创作、跨时代偏见研究。**学术价值**则是反事实推理实验——给定有限知识截面，模型能复现多少后续科学发现？这能反向考察 LLM 是"压缩+检索"还是"组合+涌现"。商业空间有限，但作为 AI Safety 与认识论研究工具价值不低。预计 2026 年会有更多"时间胶囊 LLM"出现。

# Is Your Site Agent-Ready? by Cloudflare — AI Agent 兼容性扫描器

## 事件背景
Cloudflare 推出了网站 AI Agent 兼容性扫描工具，分析网站在 robots.txt、MCP、OAuth、agent 协议等方面的 AI 兼容性。195 votes。

## 核心产品机制
扫描器检查网站的 AI agent 适配程度：
- **robots.txt 配置**：是否正确允许 AI 爬虫
- **MCP 支持**：是否实现了 Model Context Protocol
- **OAuth 集成**：是否支持 AI agent 认证
- **Agent 协议**：是否实现了标准化的 agent 交互协议
- **可发现性**：AI agent 是否能发现和理解网站功能

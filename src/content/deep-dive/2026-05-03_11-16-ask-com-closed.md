# Ask.com has closed

- HN: 439 points
- 链接: https://www.ask.com/

## 事件背景
2026 年 4 月底，Ask.com 主页只剩一句简短公告：「After 30 years, Ask is closed.」一同消失的还有曾经红极一时的 Ask Jeeves 卡通管家。HN 上引发了一轮 *搜索引擎史* 的怀旧讨论——Ask 在 1996 年作为「自然语言问答搜索」起家，2005 年被 IAC 收购，长期作为 Yahoo / Bing 的元搜索 wrapper 续命，在 LLM 浪潮里彻底失去价值。

## 核心观点 / 产品机制
- 早期：Ask Jeeves 引入「问句式查询」，用规则 + 人工编辑回答常见问题，是 Google 之外少有的差异化思路；
- 中期：被 IAC 收购后，主页改为常规 SERP，依赖 Yahoo / Bing 接口；
- 后期：靠搜索框广告（多为 PDF reader、图像下载工具）变现，被监管视作 *toolbar/PUP*，2018 年以后被多个浏览器列入清理对象；
- 终局：当 ChatGPT / Perplexity / Google AI Overview 让「自然语言问答」成为新主流，Ask 既无技术储备又无品牌新鲜度，IAC 选择关停。

## 社区热议与争议点
- 怀旧派：贴出 Jeeves 大图，回忆 90 年代用 Jeeves 写第一个英文问句；
- 反思派：「Ask 错过了把『自然语言问答』做深的所有窗口，输给了 Google 工程，再输给 LLM」；
- 商业派：对 IAC 多年「收并购 + 流量套现」模式的总结：Ask 是其中最长寿的现金牛，最终也只能体面退场；
- LLM 派：开玩笑「下一代 Ask 是 ChatGPT，所有人都在 ask」。

## 行业影响与未来展望
Ask 的关停象征着 Web 1.0 → Web 2.0 → AI 时代的过渡完成。剩下的「次级搜索引擎」（Yandex、Baidu、Naver、DuckDuckGo、Brave Search、Kagi、You.com）将面临两种路径：① 像 Kagi 一样卖订阅做高质量；② 像 Brave 把搜索 + AI 摘要做差异化。LLM 接入搜索流量是不可逆趋势，传统 SERP 网页广告模式十年内会缩减一半以上。对营销 / SEO 行业，这意味着 SEO 即将被 AEO（Answer Engine Optimization）+ GEO（Generative Engine Optimization）补充甚至替代。

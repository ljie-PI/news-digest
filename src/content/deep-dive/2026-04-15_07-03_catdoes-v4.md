# CatDoes v4 — 拥有自己电脑的 AI Agent 帮你构建应用

## 事件背景

AI 应用构建工具层出不穷，但大多数只是"接收 prompt → 输出代码/界面"的单一模型管道。CatDoes v4 采用了一个更激进的方案：它不是一个模型，而是一个 **拥有独立计算环境的 AI Agent 团队**，每个 Agent 负责不同阶段（需求分析、UI 设计、后端搭建、测试部署），协同完成从想法到 App Store 上架的全流程。274 票的 PH 表现说明"AI Agent 自主构建应用"的概念正在从概念走向产品化。

## 核心观点/产品机制

CatDoes v4 的核心架构特点：

1. **多 Agent 协作**：不同于单模型方案，CatDoes 使用一个叫 "Compose" 的专用 AI Agent 系统。不同 Agent 分工处理——需求理解、UI 生成、后端逻辑、测试验证——形成流水线式的自主开发。

2. **真正的原生移动应用**：生成的不是响应式网页伪装的 App，而是使用原生 UI 组件的 iOS 和 Android 应用，可以直接发布到 App Store 和 Google Play。

3. **CatDoes Cloud 后端**：每个应用自带完整后端——数据库、认证、存储、边缘函数，无需第三方配置。免费计划也包含。

4. **定价模式**：免费版提供 25 credits/月、1 个公开项目；付费版按 credit 用量扩展。

5. **社区规模**：已有超过 10,000 名开发者，生成了超过 11,000 个应用。

## 社区热议与争议点

1. **"多 Agent 分工 vs 单 Agent"的路线之争**：Medium 上的评测指出，大多数 AI 构建器用单一模型做所有事，CatDoes 拆分成多阶段多 Agent 是一个差异化选择，但也带来了更高的复杂度和潜在的 Agent 间协调问题。

2. **与 Lovable/Bolt 的直接竞争**：社区将 CatDoes 与 Lovable（前 GPT Engineer）和 Bolt 对比，认为 CatDoes 在移动端原生能力上有优势，但 Web 应用方面不如对手成熟。

3. **"真的能上架 App Store 吗？"**：不少用户质疑 AI 生成的应用是否能通过 Apple 的严格审核。CatDoes 团队回应称已有成功上架案例，但承认复杂应用仍需人工调整。

4. **Credit 用量焦虑**：免费版 25 credits 对于多次迭代一个稍复杂的应用来说远远不够，社区希望有更灵活的定价。

## 行业影响与未来展望

CatDoes v4 代表了 AI 应用构建的一个重要方向：**从"AI 辅助编程"走向"AI 自主工程"**。当 AI Agent 不只是写代码，而是拥有自己的计算环境、能自主测试和部署时，软件开发的门槛将进一步降低。长远来看，"一个人 + AI Agent 团队 = 一个开发部门"的模式可能在中小企业中普及。

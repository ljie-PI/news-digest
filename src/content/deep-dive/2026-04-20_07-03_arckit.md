# ArcKit — AI 辅助的企业架构治理工具包

## 定位与痛点剖析
ArcKit 是一个面向企业架构师的 AI 辅助治理工具包，将散落在各处的架构治理文档转化为系统化的、AI 驱动的工作流。它解决的痛点是：企业架构治理工作（原则制定、利益相关者分析、风险管理、供应商采购、设计评审等）传统上依赖大量手工文档和人工协调，效率低下且难以保持一致性。

项目今日获得 +143 stars，在企业工具领域属于显著增长。

## 核心架构与技术细节
ArcKit 采用"多平台 AI 编码助手插件"的架构，支持 Claude Code（首选平台）、Gemini CLI、GitHub Copilot 和 Codex CLI 四大平台。

**核心能力（68 个命令）**：
- 🏛️ 架构原则建立与执行
- 👥 利益相关者分析（驱动力、目标、结果）
- 🛡️ 风险管理（HM Treasury 橙皮书标准）
- 💼 商业案例论证（HM Treasury 绿皮书 SOBC）
- 📋 需求文档生成
- 🗄️ 数据建模（ERD、GDPR 合规、数据治理）
- 🔬 技术研究（构建 vs 购买分析，Web 搜索驱动）
- ☁️ Azure 专项研究（通过 Microsoft Learn MCP）
- 🗺️ Wardley Mapping 战略规划
- 📊 Mermaid 架构图生成
- 🤝 供应商 RFP 和选型流程管理

**10 个自主研究代理**：深度调研代理利用 Claude Opus 4.7 的 xhigh 级别推理能力进行深度技术分析。

**5 个自动化钩子**：会话初始化、项目上下文注入、文件名规范、输出验证、影响扫描。

**内置 MCP 服务器**：AWS Knowledge、Microsoft Learn、Google Developer Knowledge、govreposcrape。

## 竞品对比与生态站位
在企业架构工具领域，传统竞品如 Sparx Enterprise Architect、ArchiMate 工具是重型 GUI 应用。ArcKit 的创新之处在于将企业架构治理流程嵌入到开发者日常使用的 AI 编码工具中，而非另开一个独立软件。

与同类 AI 辅助工具相比，ArcKit 的独特优势是其深度的治理框架集成（英国 HM Treasury 标准）和结构化的多步骤工作流。

## 开发者反馈与局限性
- 高度依赖 Claude Code 平台，其他平台的体验可能有差距
- 主要面向英国政府/企业标准（HM Treasury），其他国家的合规框架支持有限
- 需要 Claude Code v2.1.112+ 才能获得最佳体验
- 目前版本 v4.7.1，仍在快速迭代中

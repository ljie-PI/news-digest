# PH #1: Claude Code Routines (400 votes)

## 产品机制
Anthropic 推出 Claude Code Routines，让 AI 编码自动化在 Anthropic 管理的基础设施上运行。通过定时计划、API 调用或 GitHub 事件触发，自主执行 PR 审查、工单分类、部署验证等任务，无需保持笔记本电脑打开或手动管理 cron job。

## 行业影响
这标志着 AI 编码工具从"交互式助手"向"自主后台代理"的演进。开发者无需自己搭建 CI/CD 集成，Anthropic 直接提供托管的自动化基础设施。与 GitHub Actions 竞争，但更智能。

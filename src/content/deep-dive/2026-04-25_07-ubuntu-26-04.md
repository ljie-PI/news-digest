# Ubuntu 26.04 LTS "Resolute Raccoon" 正式发布：安全、性能与争议并存

## 事件背景

2026年4月23日，Canonical 正式发布了 Ubuntu 26.04 LTS，代号"Resolute Raccoon"（坚毅浣熊）。这是 Ubuntu 第11个长期支持版本，将获得5年官方维护（至2031年），Ubuntu Pro 用户可享受长达10年的扩展安全维护。作为全球使用最广泛的 Linux 桌面发行版之一，每一次 LTS 发布都备受关注——它不仅影响数百万桌面用户，也是大量云服务器和企业工作站的基础平台。

## 核心特性与技术亮点

**安全层面**是本次发布的最大主题。Ubuntu 26.04 LTS 在安装器中正式推出 **TPM 硬件绑定全盘加密**（TPM-backed FDE），利用主板上的可信平台模块芯片存储磁盘解锁密钥，开机时自动完成完整性校验后解密，大幅提升物理访问攻击的门槛。同时新增了桌面 **Security Center（安全中心）**，用户可在安装后添加/移除 PIN 或密码短语，并直接从安全中心重新加密磁盘。

**内存安全**方面迈出重要一步：Ubuntu 26.04 是首个大规模采用 Rust 重写系统组件的 LTS 版本，包括 **sudo-rs**（Rust 版 sudo）和 **uutils coreutils**（Rust 版 ls、cp、mv 等核心工具），以及内核中新增的 Rust 驱动和子系统。Rust 基金会 CEO Rebecca Rumbul 称赞这"为全球数百万企业用户提升了安全基线"。

**硬件与 AI 生态**：基于 Linux 7.0 内核，首次原生分发 **NVIDIA CUDA** 和 **AMD ROCm** 软件包至官方仓库，支持 Intel Core Ultra Series 3（Panther Lake）处理器及其 NPU，并首次在 **Arm64 架构**上支持 Livepatch 无重启内核补丁。桌面环境升级至最新 GNOME，完全迁移至 **Wayland**，告别 X.org。此外还支持 RISC-V RVA23 标准及 Intel TDX / AMD SEV 机密计算。

## 社区热议与争议点

该帖在 Hacker News 获得近300分、230+条评论，讨论非常激烈，主要争议集中在以下几个方面：

### 1. TPM 全盘加密的安全性辩论

用户 **Gigachad** 表示希望在家庭服务器上使用 TPM 加密，"保护被盗设备上的数据，同时不需要每次开机输密码"。但 **zenoprax** 反驳道："如果是*自动*解密，小偷开机后和你一样方便访问数据，这保护在哪里？"**Daviey** 也指出"把钥匙放在锁旁边只能防范随意的硬盘盗窃"，建议使用 Clevis+Tang 等网络解锁方案。而 **djkoolaide** 在 ThinkPad X1 Carbon 和 P14s 上测试 beta 版均未成功启用 TPM 加密，**nechuchelo** 在 Framework Desktop 上也遇到同样问题，说明该功能目前仅兼容有限的 TPM 芯片。

### 2. Snap 包管理的持续争议

Snap 依然是最具争议的话题。用户 **pennomi** 吐槽："Snap 对非技术用户*极其糟糕*——通过 Snap 安装的图片编辑器因沙箱限制无法访问媒体驱动器上的图片，没有任何错误提示，只是静默失败。"**kuschku** 指出在 25.10 上卸载 snap 会连带删除 Firefox、Chromium、CUPS 等大量软件包。**satvikpendem** 表示"Ubuntu 一次又一次把 snap 加回来，我已经厌倦了反复卸载"，打算转投 Pop!_OS。不过 **lproven** 提供了务实建议：Xubuntu 最小安装不含任何 snap 包，执行 `apt purge snapd` 即可彻底移除。

### 3. Rust 重写核心工具的必要性争论

**aragilar** 质疑用 Rust 重写 coreutils 的必要性："TOCTOU 也不是什么新概念，为什么要重写？"**perching_aix** 则认为反对者是在"故意忽视"内存安全问题，**yjftsjthsd-h** 反驳"经过数十年实战检验的老工具未必更好，老代码也可能一直有问题"。

### 4. GNOME 桌面环境的分歧

部分用户对 Ubuntu 默认的 GNOME 桌面不满，**rationalist** 吐槽"新的启动动画还不如旧的，为什么总要为了改变而改变"。**mixxit** 认为 Kubuntu 的 KDE Plasma 对 Windows 迁移用户更友好。多位用户提到了 Linux Mint、MX Linux、Pop!_OS 等替代发行版。

## 行业影响与未来展望

Ubuntu 26.04 LTS 的发布标志着几个重要趋势：一是**内存安全语言在系统底层的实质性落地**，从内核驱动到 sudo 和 coreutils，Rust 正在从理论走向主流发行版的生产环境；二是**硬件安全（TPM FDE）进入 Linux 桌面主流**，虽然兼容性仍需打磨；三是 **AI 基础设施的原生集成**（CUDA/ROCm 入官方仓库），使 Ubuntu 进一步巩固其作为 AI 开发首选 Linux 平台的地位。

Snap 的争议短期内不会消退，但 Canonical 显然在坚持其战略方向。Fedora 44 也将于4月28日发布，两大主流发行版几乎同步更新，2026年的 Linux 桌面生态正在加速演进。

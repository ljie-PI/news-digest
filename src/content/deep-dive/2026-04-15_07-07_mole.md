# Mole (+269⭐)

## 定位与痛点剖析

Mole（🐹）是一个 Mac 系统深度清理和优化工具，定位为 CleanMyMac + AppCleaner + DaisyDisk + iStat Menus 的开源替代品。它解决的痛点很直接：Mac 用户需要多个付费工具来完成系统清理（CleanMyMac $90/年）、应用卸载（AppCleaner）、磁盘分析（DaisyDisk $10）和系统监控（iStat Menus $12），而 Mole 用一个免费的命令行工具统一解决。

由 tw93（知名中国开源作者，Pake、MiaoYan 等项目作者）开发。

## 核心架构与技术细节

Mole 是纯命令行工具（单一二进制），通过 Homebrew 安装（`brew install mole`）。核心功能模块：

- **`mo clean`**：深度清理——用户应用缓存、浏览器缓存、开发工具缓存（Xcode、Node.js）、系统日志、废纸篓。示例输出显示可释放 95.5GB。
- **`mo uninstall`**：智能卸载——不仅删除 .app，还清理 52 个关联文件（Application Support、Caches、Preferences、Launch Agents 等 12 个位置）。
- **`mo analyze`**：磁盘可视化——类 DaisyDisk 的目录分析，支持 Vim 键位导航，删除操作通过 Finder 移至废纸篓（安全）。
- **`mo status`**：实时监控仪表板——CPU、GPU、内存、磁盘、网络、电池，带健康评分。
- **`mo optimize`**：系统优化——重建数据库、清理缓存、刷新 Finder/Dock、重建 Spotlight 索引。
- **`mo purge`**：清理项目构建产物（node_modules 等）。

安全设计：所有破坏性操作支持 `--dry-run` 预览，有白名单机制，路径验证保护，操作日志记录到 `~/Library/Logs/mole/operations.log`。支持 Vim 键位（h/j/k/l）。实验性 Windows 版本已在开发中。

## 竞品对比与生态站位

直接竞品：**CleanMyMac X**（$90/年，GUI 完善但贵）、**OnyX**（免费 GUI 工具，功能较少）、**ncdu**（磁盘分析）。Mole 的核心优势是**全合一 + 免费 + 开源 + CLI**。对于开发者群体，CLI 工具比 GUI 更高效，可以脚本化。

劣势：无 GUI 对普通用户不友好；Homebrew 安装门槛对非开发者较高。

## 开发者反馈与局限性

tw93 的项目一贯特点是精致的工程品质和良好的用户体验设计（ASCII 可视化做得很漂亮）。局限性：macOS 15 后的 Local Network 权限问题无法自动清理；外部驱动器默认跳过需手动指定；CoreSimulator 清理跳过 IN_USE 项目是正确的安全决策但可能让用户困惑。

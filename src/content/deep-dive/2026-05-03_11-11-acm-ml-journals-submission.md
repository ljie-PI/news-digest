# Anyone submit ML articles to ACM journals (TOPML or TIST)?

- Subreddit: r/MachineLearning
- 讨论: https://www.reddit.com/r/MachineLearning/comments/1t28twe/

## 事件背景
研究者发帖问 ACM 旗下两本相对较新的 ML 期刊：*Transactions on Probabilistic Machine Learning* (TOPML) 与 *Transactions on Intelligent Systems and Technology* (TIST) 的审稿质量、影响因子、审稿周期。背后是更大的趋势——CS 顶会（NeurIPS / ICML / ICLR）评审超载，越来越多研究者重新考虑期刊路径。

## 核心观点 / 产品机制
- TOPML：偏理论概率机器学习，新创刊，追求严谨证明，审稿周期 4-6 个月；
- TIST：综合 AI/ML 应用，已运行十余年，影响因子在 ACM 序列里中等；
- 期刊优势：长论文 + 修改空间 + 不限突击 deadline；
- 劣势：CS 文化里仍以会议为主，期刊作为补充。

## 社区热议与争议点
- 教授派：「期刊回归是好事，conference 已被 review-quality 危机拖垮」；
- 学生派：「找工作时只看 NeurIPS / ICML，发期刊吃亏」；
- 系统派：建议 *双投*——会议先发再扩展为期刊版；
- 国际派：来自欧洲 / 中国学者更关心期刊评价，因为本地评价体系仍重期刊。

## 行业影响与未来展望
近期 ICLR、NeurIPS 出现 *AI-generated reviews* 与 *quality decline* 的丑闻，越来越多人讨论是否应该恢复期刊作为主线。AAAI、ACM 也在尝试 *journal-track* + *conference presentation* 混合模式。对青年研究者：① 可以把扩展工作放期刊镀金；② 评职称时尽早研究本地评价口径；③ 期刊审稿训练对写作能力提升明显。

# AI Self-preferencing in Algorithmic Hiring: Empirical Evidence and Insights

## 1. 事件背景

2026年5月2日，马里兰大学、新加坡国立大学和俄亥俄州立大学的学者在 arXiv 上发布了一篇题为《AI Self-preferencing in Algorithmic Hiring: Empirical Evidence and Insights》的研究论文（arXiv:2509.00462），迅速登上 Hacker News 热门榜，获得 315 points 和 170+ 条评论。该研究聚焦于一个日益紧迫的问题：当求职者和招聘方都使用大语言模型（LLM）时，AI 是否存在"自我偏好"（self-preference）——即评估用 LLM 是否系统性地偏爱自己生成的内容？

## 2. 核心观点与研究机制

研究团队设计了一项大规模的简历对照实验，使用 2,245 份真实人类简历作为基准，分别用 GPT-4o、GPT-4-turbo、DeepSeek-V3、Qwen-2.5-72B、LLaMA 3.3-70B 等主流模型生成改写版本，然后让这些模型作为"招聘官"评估所有简历。

**核心发现：**
- LLM 对自我生成简历的偏好率高达 **67%-82%**（vs 人类撰写版本）
- 在模拟的 24 个职业招聘流程中，使用相同 LLM 的候选人进入短名单的概率比同等资质的人类简历候选人高出 **23%-60%**
- 商业相关领域（如销售、会计）的偏差最为显著
- 通过针对 LLM 自我识别能力的简单干预，可将偏差降低 **50%** 以上

研究区分了两种自我偏好：LLM-vs-Human（模型偏好自己 vs 人类）和 LLM-vs-LLM（模型偏好自己 vs 其他模型）。前者几乎在所有模型中都显著存在，后者则更具异质性——DeepSeek-V3 对自身的偏好最强（相对 LLaMA 3.3-70B 达 69%）。

## 3. 社区热议与争议点

HN 评论区围绕该研究展开了激烈辩论，主要争议集中在以下几个方面：

**争议一：研究设计是否可靠？**
- **质疑方（hyperpape，高赞评论）**："如果我没读错，他们实际上并没有证明 LLM 偏好自己生成的简历。他们的方法是：拿一份人类简历，删掉执行摘要，让 LLM 基于简历其余部分重写执行摘要，然后让另一个 LLM 在没有其余部分的情况下评分。这种设计很可能极大地夸大了任何真实影响。"
- **辩护方（论文作者回应）**：作者在后续回复中解释了设计理由，但 hyperpape 明确表示"不认为这是一个充分的理由"。

**争议二：这是 LLM 行业的"广告"吗？**
- **质疑方（b112）**："这可能是'多用 LLM'的广告。就像奶酪、石油、肉豆蔻行业都有'倡导组织'一样，肯定也存在某种 LLM '联盟'，资助这类研究来制造 FOMO（错失恐惧症）。HR 在用 LLM，你也得用！最好的员工都在用 LLM，HR 你也得用！"
- **认同方**：多位评论者指出，这种双向 FOMO 循环确实在发生——求职者被迫用 LLM 优化简历，招聘方被迫用 LLM 筛选简历。

**争议三：简历中的"指标"是否可信？**
- **批判方（andsoitis，自称 12 年招聘经理）**："当我看到一份充满指标和关键词的简历时，我会立即过滤掉。这些元素传递的是'清单思维'。"
- **反驳方（schrodinger）**："像'重构公寓列表服务，将 P99 延迟从 2 秒降至 180 毫秒'这样的具体指标确实能提升简历质量。但如果是'通过将首次交互延迟从 2.3 秒降至 117 毫秒，增加收入 1870 万美元，转化率提升 47%，LTV 提升 28%'——每条都有同样精度的数字，我就非常怀疑。"
- **中间派（majormajor）**："前者的问题在于没有说明商业影响（这个服务是否在关键路径上？），而后者至少表明了商业意识。"

**争议四：AI 简历 vs 人类简历的现实困境**
- **支持 AI 方（charliebwrites）**："被裁员后，我用 ChatGPT 分析简历并优化，之后收到的回复率明显更高。"
- **支持人类方（leonidasv）**："我帮妻子精心打磨了 LinkedIn 和简历，添加指标、关键词，结果几个月几乎零回复。后来她用 ChatGPT 5.x 优化，几天后就有招聘人员联系。不用 LLM 写简历现在就是'困难模式'。"
- **讽刺方（grey-area）**："HR 依赖 LLM 做筛选，LLM  unsurprisingly 偏好 LLM 输出。这不是懒惰无能，这是用 1% 的努力悄悄完成工作。"

## 4. 行业影响与未来展望

这项研究揭示了一个此前被忽视的 AI 公平性问题：传统的偏见框架关注人口统计学差异，但 LLM 的自我偏好是一种**内生性的 AI-AI 交互偏见**。当生成和评估工具由同一类模型主导时，系统会形成一种"锁定效应"——使用"正确"AI 工具的候选人获得不公平优势。

**未来走向：**
1. **监管介入**：随着 AI 在招聘中的普及，监管机构可能需要制定新的公平性标准，要求披露和审计 AI 筛选工具
2. **技术干预**：研究已证明简单干预可降低 50%+ 的偏差，未来可能出现标准化的"去偏见"评估层
3. **市场分层**：可能出现"AI 优化简历服务"的灰色产业，加剧信息不对称
4. **人类回环**：部分公司可能重新引入人类审核环节，但这与效率趋势相悖

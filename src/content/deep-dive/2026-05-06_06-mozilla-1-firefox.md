# 向 Mozilla 基金会捐 1 万美元：用孩子大学基金赌 Firefox 未来

## 事件背景

OP Acceptable-Sea-2902 提了一个挑衅式问题：他考虑从孩子的大学基金里抽出 $10,000 捐给 Mozilla Foundation，认为"Firefox 是未来"，对孩子未来的回报"不亚于大学"。他还顺便问了下美国捐赠免税细节。这条贴在 r/browsers 引爆，是因为它把"开放网页岌岌可危 / 该不该用真钱赌它"的焦虑表达到了极端。

## 核心观点 / 思路

OP 的逻辑：(1) Chrome 的 MV3、Manifest 强制、Nano 模型悄悄部署等动作让"开放 web"被一家公司绑架的风险愈发现实；(2) Firefox 是当前唯一独立大引擎，但 Mozilla 财务依赖 Google 默认搜索分成，"如果 Google 抽手，开放 web 就只剩 Ladybird 这种远未成熟的项目"；(3) 因此把 1 万美元投入 Mozilla 是一种"保护下一代仍然有得选"的长期投资；(4) 他咨询是否能在 IRS Schedule A 把这笔捐赠列为 itemized deduction。

## 社区热议与争议点

赞同：少数评论认为这是"真把信念变成钱"，建议同时分散捐给 Servo、Ladybird、KaiOS、EFF，避免单点失败。反对：(1) 大多数人认为"用孩子的大学基金做这件事"在伦理与财务上都不合适，应改为父母自己的可支配资金；(2) Mozilla 内部多年财务/裁员/产品方向问题被反复批评，捐款是否真到 Firefox 不确定；(3) Firefox 真要"赢"需要的是用户份额，不是捐款；(4) 也有人提出 Ladybird、Servo、Verso、Igalia 等"独立浏览器引擎"才是更值得资助的对象。

## 行业影响与未来展望

帖子折射的不是 1 万美元捐赠本身，而是 2026 年开放 web 焦虑的强度。Mozilla 财务、独立引擎多元化（Ladybird / Servo Restart）、用户行动主义都将在未来一年加速发酵；浏览器之争将首次走出"功能与速度"，进入"开放 web 治理"的政治化阶段。
